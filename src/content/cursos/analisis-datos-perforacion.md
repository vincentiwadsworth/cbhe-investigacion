Curso práctico sobre la aplicación de técnicas de ciencia de datos y machine learning para optimizar operaciones de perforación. Los participantes trabajarán con datasets reales del sector para construir modelos predictivos de rendimiento.

## Módulos

1. Fundamentos de Python para análisis de datos de perforación
2. Limpieza y preparación de datos operacionales
3. Modelado predictivo de ROP (Rate of Penetration)
4. Detección temprana de eventos de perforación (kicks, pérdidas)
5. Visualización y dashboards para toma de decisiones en tiempo real

## Requisitos

Conocimientos básicos de programación. Experiencia en operaciones de perforación. Notebook personal con Python 3.10+ instalado.
